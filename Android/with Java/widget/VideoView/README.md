# VideoView
## Objective
In this article, I will teach you how to use VideoView to make a small app (which is similar to YT)

It can

      1. select a video from file picker.
      2. play, stop, resume, replay the video.
      3. set the speed.
      4. set the volume, mute, and unmute.
