# getChildAt -- Get the item of the ViewGroup with given index.
## intro
Given index, return the corresponding item of the ViewGroup.

## syntax
    View getChildAt(int index);
    
### parameter

In View getChildAt(int index),

    index : the index.

### return value
    View
