# LayoutInflater.from
## intro
Get LayoutInflater from the given Context
## syntax
    static LayoutInflater LayoutInflater.from(Context context)
### static LayoutInflater LayoutInflater.from(Context context)
#### parameter
    context : A context to LayoutInflater class.
#### return value
    LayoutInflater
