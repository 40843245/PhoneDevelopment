# ActivityResultCallback
## Objective 
In this article, I will fully discuss the class ActivityResultCallback.

## Override method (method that uninstantiated method that can be overidden)
### onActivityResult
#### intro
A callback function that is called when the activity finishes.
#### syntax
    public void onActivityResult(ActivityResult result);
#### parameter
    result : It refers the result of activity. It should be ActivityResult type. 
#### return value
    None, void type
